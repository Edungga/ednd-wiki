You cast this spell on an object no more than 1 foot on a side, doubling the object's perceived value by adding illusionary flourish or reducing its perceived value by half with the help of illusionary dents and scratches. Anyone examining the object must roll an INT (Investigation) Check.

**At Higher Levels**. The the size of the object increases by 1d6 for each spell slot level above 1.
